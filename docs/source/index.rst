Welcome to UP4ROS's documentation!
===================================

**UP4ROS** is a ROS wrapper for the AIPlan4EU Unified Planning library available 
at `this link <https://github.com/aiplan4eu/unified-planning>`_.
This wrapper allows to call the unified planning library and solve planning problems described using PDDL.
The source code of this ROS repository can be found at the following link.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation`. 
For an overview of the services and actions exposed by the main node of the package, please see  :ref:`up4ros`.

.. note::

   This project is under active development.
